# Caixa Eletronico

Sistema simples de caixa eletronico criado no Debian usando o editor vi.

## Arquivo principal

caixa_eletronico

## Como compilar no Debian

gcc -x c caixa_eletronico -o caixa_eletronico_app

## Como executar

./caixa_eletronico_app

## Funcionalidades

- Verificar saldo
- Depositar valor
- Sacar valor
- Pedir emprestimo
- Pagar emprestimo
- Consultar emprestimo
- Sair
