# Caixa Eletronico

Sistema simples de caixa eletronico criado no Debian usando o editor vi.

## Arquivo principal

caixa_eletronico.c

## Como compilar no Debian

gcc caixa_eletronico.c -o caixa_eletronico

## Como executar

./caixa_eletronico

## Funcionalidades

- Verificar saldo
- Depositar valor
- Sacar valor
- Pedir emprestimo
- Pagar emprestimo
- Consultar emprestimo
- Sair
